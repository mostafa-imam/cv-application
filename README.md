# CV Application

A CV builder application built with React and Vite, created as part of The Odin Project curriculum

## 🚀 About The Project

This is a dynamic CV Builder application built from scratch using React. I created this project as part of The Odin Project curriculum ([specifically this lesson](https://www.theodinproject.com/lessons/node-path-react-new-cv-application)) to practice React core concepts - especially managing state, handling forms, and passing data between custom components via props.

The app features a split-screen design where users can input their general info, work experience, and education details on one side, and see a formatted resume update in real-time on the other.

## 🛠️ Built With

- React (Vite)
- HTML5 & CSS3 (Flexbox/Grid layout)
